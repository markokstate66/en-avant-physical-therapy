Set Up Stripe (Payments)

Stripe handles all credit card payments for your services.

1.1 Create a Stripe Account
  1. Go to stripe.com/register
  2. Enter your email and create a password
  3. Verify your email address
  4. Complete the business verification (you'll need):
    • Business name and address
    • EIN or SSN (for tax purposes)
    • Bank account for deposits
Important: Complete the full verification to accept real payments. Until verified, you can only use test mode.
1.2 Create Payment Links

You need to create a payment link for each service:

Service Name Price Your Payment Link
Initial Evaluation (60 min) $200.00
Follow-Up Session (45 min) $150.00
Extended Session (60 min) $175.00
Home Visit (60 min) $225.00
How to create a payment link:
  1. In Stripe Dashboard, go to Payment Links (left sidebar)
  2. Click "+ New"
  3. Select "Products or subscriptions"
  4. Click "+ Add new product"
  5. Enter the service name and price from the table above
  6. Set as One-time payment
  7. Click "Create link"
  8. Copy the link and paste it in the table above
  9. Repeat for each service
Tip: You can customize the checkout page with your logo and colors in Payment Link settings.

Set Up Calendly (Scheduling)

Calendly allows patients to book appointments directly on your website.

2.1 Create a Calendly Account
  1. Go to calendly.com/signup
  2. Sign up with your email or Google account
  3. Complete the onboarding wizard
2.2 Set Up Your Availability
  1. Go to Availability in the left menu
  2. Set your working hours (e.g., Mon-Fri 8am-6pm, Sat 9am-2pm)
  3. Add any buffer time between appointments
  4. Set how far in advance people can book
2.3 Create Event Types

Create event types for each appointment type:

Event Name Duration
Free Consultation 30 minutes
Initial Evaluation 60 minutes
Follow-Up Session 45 minutes
Extended Session 60 minutes
Home Visit 60 minutes
2.4 Get Your Calendly Link
  1. Click on your profile icon (top right)
  2. Your Calendly link looks like: https://calendly.com/your-username
  3. Copy this link:
Calendly URL
Tip: Connect Calendly to your Google or Outlook calendar to automatically block off busy times.
